# retailops_core

This package wrapper represents the shared platform capabilities of the monorepo.

## Intended scope

The package is the natural future home for common settings, shared schemas, repository abstractions, utilities, and generic service helpers that are currently developed inside `core/`.

## Why it exists now

The roadmap calls for a modular repository from the beginning. This wrapper keeps the project ready for a later split into separately published packages without forcing that split too early.
