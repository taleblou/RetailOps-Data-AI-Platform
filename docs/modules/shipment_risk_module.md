# Shipment Risk Module

Shipment risk scores open orders for delay probability and prioritizes manual follow-up.

## Inputs

Order, shipment, promised-date, fulfillment, and inventory context from trusted marts.

## Outputs

Delay-risk summaries, order detail views, and action-oriented artifacts for operations teams.
