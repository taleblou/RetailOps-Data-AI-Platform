# Returns Intelligence Module

Returns intelligence estimates return risk and driver explanations to support fraud, policy, and customer-experience decisions.

## Inputs

Customer history, order context, payment signals, product characteristics, and delivery outcomes.

## Outputs

Return-risk summaries, order-level explanations, and artifacts that can feed business-review or policy workflows.
