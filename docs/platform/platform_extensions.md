# Platform Extensions

Platform extensions widen the repository beyond batch ingestion and mart-centric workflows.

## Included services

- CDC
- streaming
- lakehouse
- metadata and lineage
- feature store
- query federation
- advanced serving

## Design approach

These modules are optional. They reuse the same repository conventions and artifact locations as the core platform, and they now expose deployment-ready bundle generators, compose summaries, readiness checks, and generated companion files for operators.
